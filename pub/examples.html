In this example an entire song sheet with popup and chord section options on is generated.

Some features of the library demonstrated in this example are:

  • show/hide toggle for the chords section which shows/hides the section when clicked on
  • some chords with multiple shapes displayed as carousel such as the G chord (click on the small squares above the chord in order to switch between the chord shapes).
  • Barre chords (e.g. the F chord), chords starting at fret numbers > 0 (e.g. Am chord) and chords with mute/open strings (e.g. Em)
  • Popup display of chords in the song content portion: hover over the chord names and the chord shape will be displayed.
  • Option to switch between popup and regular display: Check or uncheck the checkbox display chords as popups.

This example is created using the following code:

'use strict';

const song_info = new GeneralSongInfo('Flying', 'Anathema', 'Hindsight', 'eadgbe', 4, 'c');
const song_container = document.createElement('div');
song_container.setAttribute('style', 'display: flex; justify-content: center;')
const Am = new Chord('Am', [-1, 0, 2, 2, 1, 0]);

const F = new Chord('F', [1, 3, 3, 2, 1, 1], {1: [1, 6]});

G.add_shape([3, 2, 0, 0, 3, 3]);
const verse1 = new LyricChordGroup(

const body = document.querySelector('body');
